# Shlok's Portfolio Website

Hi! This is **my portfolio website** where I showcase my skills, hobbies, and projects. I built it using **HTML, CSS, and JavaScript**, and added some interactive stuff to make it more fun and user-friendly.

---

## Technologies I Used

* **HTML** – for the structure of the site
* **CSS** – for styling, layout, and responsiveness
* **JavaScript** – for the interactive parts like my hobby list and contact form
* **Google Fonts (Montserrat)** – because I wanted a clean, modern look

---

## How My Website Is Structured

1. **Header**

   * Shows my name and a tagline: “Learn. Improve. Grow.”
   * Centered with some padding and a nice background color.

2. **Navigation Bar**

   * Links to Home, About, Projects, and Contact sections.
   * Clicking the links scrolls smoothly to the right section.

3. **Home Section**

   * A short intro about me.
   * Shows a list of my hobbies.
   * You can add new hobbies using the input box and “Add” button.
   * Every hobby also has a delete button so you can remove it.

4. **About Section**

   * A short paragraph about me and what I like.

5. **Projects Section**

   * Shows my projects in neat cards.
   * Each card has a title and a hover effect for some interactivity.
   * Uses Flexbox so the layout looks good on all screen sizes.

6. **Contact Section**

   * A simple contact form with Name, Email, and Message fields.
   * When you submit it, it shows a confirmation alert.

7. **Footer**

   * Links to my **LinkedIn and GitHub**.
   * Shows the current date and time that updates automatically.

---

## Features I Added

* **Responsive design** – works on phones and desktops
* **Interactive hobby list** – add and remove hobbies easily
* **Live date and time** – updates every second in the footer
* **Project cards** – neat layout with hover effects
* **Contact form** – simple and functional

---

## Styling

* **Fonts**: Montserrat for a clean, modern look
* **Colors**: Purple-themed for header, navbar, buttons, and cards
* **Sections**: Card-like containers with rounded corners and shadows
* **Buttons**: Hover effects for better interactivity
* **Responsive**: Adjusts layout and navbar on smaller screens
